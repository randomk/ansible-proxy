# Ansible-proxy
Ansible Project to install a proxy server

## Requirements
 * Ansible
 * Python
 * SSH
 * Git
 * sudo yum install -y libselinux-python

## Usage

1. Clone this repo
2. Initialize sub-modules:
    * Inside dir of cloned repo, run:  
        *   `git submodule update --init`
3. Run script  
    * `./run.sh`
